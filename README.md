# Sistema de Controle de Estoque

Sistema web desenvolvido para controle de estoque de produtos, com cadastro, edição e listagem.

## Tecnologias utilizadas

- Node.js
- Express
- MySQL
- HTML
- CSS
- JavaScript

## Funcionalidades

- Cadastro de produtos
- Listagem de produtos
- Atualização de estoque
- Remoção de produtos
- Integração frontend + backend via API

## Estrutura do projeto

/backend → API Node.js  
/frontend → Interface do sistema  

## Como rodar o projeto

1. Clone o repositório

git clone https://github.com/maarlond/estoque-namorada

2. Instale as dependências

cd backend  
npm install  

3. Inicie o servidor

node server.js

4. Abra o frontend no navegador
